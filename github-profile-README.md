## PopHealth Observatory (Profile Overview)

![Data Analysis Process Flowchart](docs/assets/images/data_analysis_flowchart.png)

Population health & nutrition analytics toolkit (NHANES focus) providing:
- Resilient multi-pattern NHANES XPT downloads
- Harmonized demographic & anthropometric slices
- Metadata manifest generation with schema versioning
- Experimental pesticide Retrieval-Augmented Generation (RAG) scaffold

Key Pillars: Acquisition • Harmonization • Derivation • Retrieval • Insight

Status: Active development; expanding component loaders & quality tooling. R layer (S4 + Arrow) planned—no R source yet.

See full project README for usage and roadmap.
